# Rutracker keepers widgets

A set of widgets for rutracker.org keepers 


## Install

&ast; Node.js (javascript runtime) and Tampermonkey (browser ext) should be installed

```
1) npm install

2) npm run dev

3) npm run us:upload // this will upload userscript to your browser

4) // then add krs_url property to userscript config inside userscript

5) // then open rutracker.org and endjoy development 
```
